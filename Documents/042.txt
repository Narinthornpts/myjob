Responsibilities:

Designing and implementing Java-based applications

Analyzing user requirements to inform application design
วิเคราะห์ความต้องการของผู้ใช้เพื่อความต้องการออกแบบแอพพลิเคชั่น

สามารถทำงานกับเพื่อนร่วมทีมได้

Requirements:
คุณสมบัติที่ต้องการ

At least Bachelor’s Degree in Computer Science, Computer Engineering, Software Engineer, IT or related fields.
ต้องการวุฒิขั้นต่ำ วุฒิปริญญาตรี ในสาขาวิทยาการคอมพิวเตอร์, วิศวกรรมคอมพิวเตอร์, วิศวกรซอฟต์แวร์, ไอทีหรือสาขาที่เกี่ยวข้อง

Proficient in Java and experience in developing web applications using web framework (JSF,  Spring MVC, Spring Boots)
มีความเชี่ยวชาญใน Java และประสบการณ์ในการพัฒนาเว็บแอปพลิเคชันโดยใช้งานเว็บ (JSF, Spring MVC, Spring Boots)
